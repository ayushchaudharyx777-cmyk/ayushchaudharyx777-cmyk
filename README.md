*A Python-based personal AI dashboard inspired by Tony Stark's assistant. Built from the ground up as a comprehensive, long-term learning project.*

<details>
  <summary><b>🛠️ View Technical Features & Integrations</b></summary>
  <br>
  <ul>
    <li>🖥️ <b>Interface:</b> Custom GUI built with CustomTkinter, featuring an animated startup sequence.</li>
    <li>🧩 <b>Intelligence:</b> Local LLM powered by Ollama (llama3.2) for rapid, secure, and offline AI responses.</li>
    <li>🗣️ <b>Voice Ops:</b> Seamless voice interaction via speech recognition combined with pyttsx3/ElevenLabs TTS.</li>
    <li>🎵 <b>Integrations:</b> Spotify control, OpenCV webcam vision, Flask LAN server, and librosa voice emotion analysis.</li>
